第二篇文章 - 個人網站PO文練習

By Iris 21 December 2021

這是我的第二篇文章

PO文步驟

第一步

在 content 內的 blog 資料夾裡，新增一個 markdown 檔案，例如： XXX.md

第二步

加入 hugo 文章格式設定 ※ 一定要放在最前面

依內容修改標題、時間、照片…

第三步

撰寫文章內容，完成後存檔

第四步

回到終端機(cmd)上，輸入 hugo 產生程式碼

第五步

回到 exampleSite 資料夾，就會發現 public 資料夾裡又更新了很多檔案和資料夾

第六步

將 public 資料夾內的東西全部複製，然後貼到 clone的 repo 資料夾(github.io)裡面，並全部取代

第七步

回到 Github Desktop 上，會發現有許多改動的檔案，加上 commit 訊息之後就可以 commit 了

第八步

再將這個 commit 上傳到 Github

第九步

回到個人網站上就會看到新PO的文章囉！

(如果還沒出現，就多等幾分鐘或多重新整理幾次就會看到了)
